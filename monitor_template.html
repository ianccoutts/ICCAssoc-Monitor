Missing Fields Analysis — Historical Deals
Based on sampled closed/dead deals, these fields were commonly missing, limiting historical analysis:
  • Property Address — missing ~60%; prevents geographic/market analysis
  • Lender Name — missing ~70% of closed deals; blocks lender performance tracking
  • Deal Description — missing ~80%; limits post-mortem analysis
  • Loan Type / Property Type — not a native HubSpot field; inferred from name only
  • Borrower Name — not a native field; buried in description if present at all
  • Amount — present in most deals ✓
  • Close Date — present in most deals ✓
Recommendation: Add custom HubSpot fields for Borrower Name, Property Type, and Lender Name as required fields going forward to enable lender performance, market penetration, and win/loss analysis.
